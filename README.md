# Taller EIV 2026  
**Autor:** Roque Alfaro Navarro  

Este repositorio contiene los ejercicios desarrollados en el curso  
**“Ciencia de Datos para la Programación de Servicios de Salud”**  
Escuela de Medicina – Escuela Internacional de Verano (EIV), 2026.

## Contenidos - Talleres del curso
Ejercicio 0: Anonimización y seudoanonimización de datos en salud
Ejemplos prácticos de resguardo de datos sensibles y protección de la confidencialidad.

Ejercicio 1: Extracción de datos (Data Extraction)
Carga de datos crudos desde distintas fuentes y formatos (parquet, Excel).

Ejercicio 2: Normalización de datos en salud
Limpieza, estandarización y estructuración de bases para análisis posterior.

Ejercicio 3: Construcción de un cubo multidimensional
Diseño de modelos estrella, tablas fact y dimensiones para análisis de brechas en salud.

Ejercicio 4: Visualizaciones y reportes analíticos
Análisis exploratorio, series de tiempo y visualización de brechas de oferta y demanda.

Ejercicio 5: Flujo ETL aplicado a interconsultas en salud pública
Extracción, transformación y carga de datos administrativos para responder preguntas de gestión sanitaria.

Los notebooks fueron desarrollados y ejecutados en Google Colab, y versionados en GitHub para asegurar reproducibilidad.
